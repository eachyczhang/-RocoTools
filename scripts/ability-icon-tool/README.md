# 特性图标处理工具

将截图中的特性图标处理为项目标准格式（128x128 透明底圆形 PNG）。

## 目录结构

```
ability-icon-tool/
├── input/          ← 将截图放入此目录
├── output/         ← 处理后的标准图标输出到此目录
├── process.js      ← 处理脚本
└── README.md
```

## 标准格式

参照 `data/public/pets/abilities/` 中的现有特性图标：
- **尺寸**：128 × 128 px
- **格式**：PNG（RGBA 4通道，透明底）
- **形状**：圆形裁切（圆形区域外为透明）

## 使用方法

1. 将图片放入 `input/` 目录（支持 PNG、JPG、WebP）
2. 运行脚本：
   ```bash
   cd scripts/ability-icon-tool
   node process.js
   ```
3. 处理后的图标输出到 `output/` 目录，文件名与输入文件相同（统一为 .png 后缀）

## 智能模式

脚本会根据输入图片的宽高比自动判断处理模式：

| 模式 | 触发条件 | 处理方式 |
|------|----------|----------|
| **直接模式** (direct) | 图片宽高比在 0.7~1.4 之间（近似正方形） | 直接缩放为 128x128 + 圆形蒙版 |
| **提取模式** (extract) | 图片宽高比超出上述范围（如竖屏截图） | 自动检测上方图标区域 → 裁切 → 缩放 + 圆形蒙版 |

### 支持的输入类型

- ✅ 已裁切好的图标图片（正方形）→ 直接处理
- ✅ 游戏内特性详情页截图（竖屏）→ 自动提取图标

## 处理逻辑（提取模式）

1. 分析截图上方 45% 区域的亮度分布
2. 通过行/列亮度对比，定位图标的边界
3. 裁切出正方形图标区域
4. 缩放为 128x128 并应用圆形蒙版
5. 输出为 PNG 格式

## 依赖

使用项目已有的 `sharp` 库（位于 `app/server/node_modules`），无需额外安装。
