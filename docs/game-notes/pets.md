# 精灵系统

## 种族值

6 维属性：
- HP（生命值）
- 物攻
- 物防
- 魔攻
- 魔防
- 速度

## 等级

- 等级上限：**60 级**

## 个体值（天赋/资质）

个体值范围 **0~10**，捕捉或孵化时系统随机给六维中的 **三项** 分配点数，决定精灵定位。

### 综合评价 — 天分等级（由高到低）

| 天分等级 | 条件 | 说明 |
|----------|------|------|
| 了不起天分 | 三项核心个体值均 ≥7，且性格加成属性在三项中 | 最高评价，主力首选 |
| 相当好天分 | 三项核心个体值均 ≥7 | 个体值优秀但性格未匹配，适合大部分战斗 |
| 还不错天分 | 只有 **两项** 有个体值加成 | 中等水平，过渡期使用 |
| 一般般天分 | 只有 **一项** 有个体值加成 | 加成低，不建议优先培养 |

> **判定逻辑**：
> 1. 先看有几项有个体值加成：三项有加成 → 进入“了不起天分/相当好天分”判定；两项有加成 → 还不错天分；一项有加成 → 一般般天分
> 2. 三项均有加成且 ≥7 时再看性格：性格 **增加** 的属性落在个体值三项中 → 了不起天分；否则 → 相当好天分
> 3. 性格匹配只看增加属性，不要求减少的属性不在三项中
#### 示例

假设一只精灵个体值分配在 **物攻、物防、速度** 三项：

**了不起天分 vs 相当好天分**（三项个体值均 ≥7）：

| 性格 | 效果 | 天分等级 | 原因 |
|------|------|----------|------|
| 固执 | +物攻 -魔攻 | ✅ 了不起天分 | 加成的"物攻"在三项中 |
| 勇敢 | +物攻 -速度 | ✅ 了不起天分 | 加成的"物攻"在三项中（减少的速度虽在三项中，不影响判定） |
| 急躁 | +速度 -物防 | ✅ 了不起天分 | 加成的"速度"在三项中 |
| 保守 | +魔攻 -物攻 | ❌ 相当好天分 | 加成的"魔攻"不在三项中 |
| 温顺 | 无加成无减少 | ❌ 相当好天分 | 无属性加成，不满足匹配条件 |

**还不错天分 vs 一般般天分**（个体值项数不足）：

| 个体值情况 | 天分等级 | 原因 |
|----------|----------|------|
| 物攻=9, 速度=8, 物防=0 | 还不错天分 | 只有两项（物攻、速度）有个体值加成 |
| 物攻=10, 速度=0, 物防=0 | 一般般天分 | 只有一项（物攻）有个体值加成 |

### 完美奖牌

- 当三项核心个体值 **全部为 10**（满值）时，精灵会获得一个 **完美奖牌**

### 个体值与性格搭配策略

个体值和性格呈 **乘法叠加**，必须合理搭配才能发挥最大战力：

| 定位 | 个体值优先分配 | 推荐性格 |
|------|-------------|----------|
| 物理输出型 | 物攻、物防、速度 | 固执（+物攻 -魔攻）/ 勇敢（+物攻 -速度） |
| 魔法输出型 | 魔攻、魔防、速度 | 保守（+魔攻 -物攻）/ 冷静（+魔攻 -速度） |
| 肉盾/辅助型 | 生命、物防、魔防 | 加防御/生命 且 减攻击的性格 |

### 关键机制

- 捕捉/孵化时随机决定三项个体值分配
- 个体值影响精灵最终属性成长
- 性格加成与个体值乘法叠加，搭配不当会浪费个体值优势
- 个体值加成可通过 **道具** 提升
- **命定花种精灵**必然拥有三个天分加成（即必然为「相当好天分」或「了不起天分」）

## 亲密度

- 精灵最高拥有 **五级亲密度**
- 亲密度通过与精灵互动、战斗等方式提升
- 命定花种精灵捕捉后**必定自带三级亲密度**

## 多形态

- 同一精灵编号可拥有多个形态
- 每个形态独立存储，通过 UID 区分（如 pet_040_1 ~ pet_040_7）

## 进化

- **等级进化**：达到指定等级自动进化
- **特殊条件进化**：满足特定条件（文字描述）

## 特性

- 每只精灵拥有一个特性
- 特性包含：名称、描述、图标

## 性格

- 共 30 种性格
- 每种性格有：增加属性（stat_up）、减少属性（stat_down）
- **加成与减益幅度**：性格对属性的加成与减益均为 **10%**（即 +10% / -10%），具体计算公式后续补充
- 含 5 个子性格描述

## 蛋组

- 共 15 种蛋组
- 精灵可属于多个蛋组

## 精灵标签

精灵可配置以下特殊标签，用于分类和标识：

| 标签 | 说明 |
|------|------|
| 最终形态 | 处于进化链末端的精灵（可自动检测或手动配置） |
| 传说精灵 | 赛季主推的传说级精灵 |
| 赛季精灵 | 当赛季奇遇可捕捉的精灵 |
| 通行证精灵 | 通过赛季通行证获取的精灵 |
| 首领形态 | 该精灵本身是首领形态（不展示图鉴课题） |
| 拥有首领形态 | 该精灵可通过【进化之力】进化为首领形态 |
| 异色精灵 | 配置了异色立绘的精灵（自动识别，仅在异色 Tab 时显示标签） |

> 用户端精灵详情页中，标签以彩色 badge 形式显示在蛋组标签下方。

### 异色显示控制

- 管理端可通过 `show_shiny` 开关控制用户端是否展示异色内容
- `show_shiny = 1`（默认）：用户端可见异色立绘和异色筛选
- `show_shiny = 0`：用户端隐藏异色，异色筛选不返回该精灵
- 用户端筛选"异色精灵"时，展示所有形态（不限于默认形态）

## 最终形态

- 处于进化链最后一个阶段的精灵，自动标记为"最终形态"
- 判定规则：出现在某条进化路线的末尾，且**不在任何路线的中间阶段**
- 可通过管理端手动开启/关闭
- 最终形态精灵拥有额外的图鉴课题任务

## 首领形态

- **首领形态**：精灵本身就是首领形态，不展示图鉴课题
- **拥有首领形态**：精灵可通过道具【进化之力】进化为首领形态
- 两者互斥：一只精灵不会同时是"首领形态"和"拥有首领形态"

## 身高体重

- 身高单位：米（m），体重单位：千克（kg）
- 支持范围配置：最低值 ~ 最高值（如 1.50~2.15m）
- 保留两位小数
- 存储格式：`"min-max"`（如 `"1.50-2.15"`），相同值时为 `"1.50-1.50"`
- 用户端显示：范围不同时显示 `1.50~2.15m`，相同时只显示 `1.50m`

## 图鉴课题

图鉴课题是精灵的收集/养成任务系统，每只精灵拥有一组默认课题，管理员也可手动添加自定义课题。

### 默认课题规则

| 课题 | 适用范围 | 条件 |
|------|----------|------|
| 捕捉1只精灵 | 所有精灵 | — |
| 捕捉1只了不起天分的精灵 | 所有精灵 | — |
| 使精灵成功进化1次 | 非最终形态 | 最终形态开启后自动移除 |
| 获得【命定勇者】奖牌 | 最终形态 | 最终形态关闭后自动移除 |
| 捕捉一只炫彩突变的精灵 | 最终形态 | 最终形态关闭后自动移除 |
| 捕捉一只异色突变的精灵 | 最终形态 + 有异色 | 需配置异色立绘 |
| 使用【进化之力】，将精灵进化为首领形态 | 拥有首领形态 | 需配置"拥有首领形态"标签 |

### 特殊规则

- **首领形态精灵**（`is_boss_form=1`）不展示图鉴课题
- 默认课题使用统一的"精灵"称谓（不使用具体精灵名），避免多形态精灵产生歧义
- 默认课题可在管理端**手动配置是否显示**（hidden 状态），配置后不会被后续保存覆盖
- 自定义课题（手动添加的）与默认课题互不干扰

### 课题显示位置

用户端精灵详情页中，图鉴课题模块位于**属性克制关系**和**打击面分析**之间。

### 技能类型课题展示

技能类型课题在用户端以信息卡片形式展示，包含：
- 课题标题：格式为"使用{N}次{技能名}"（默认 N=2）
- 技能图标（优先专属图标，fallback 属性图标）
- 属性标签（带图标和颜色）
- 类型标签（物攻/魔攻/防御/状态）
- 学习等级（从精灵升级技能列表中获取，课题技能一定是升级学习）
- 能耗和威力

## 星级与成长值

### 普通星级（0~5星）

精灵初始为 **0 星**，通过养成材料升星，最高升至 **5 星**。

普通星级影响精灵的六维 **成长值**：

| 星级 | 生命成长值 | 其他五维成长值（各） |
|------|-----------|-------------------|
| 0星 | +0 | +0 |
| 1星 | +20 | +10 |
| 2星 | +40 | +20 |
| 3星 | +60 | +30 |
| 4星 | +80 | +40 |
| 5星 | +100 | +50 |

> **规律**：每升一星，生命成长值 +20，其他五维（物攻/魔攻/物防/魔防/速度）成长值各 +10。

### 觉醒星级（0~5星）

普通五星之后可继续升星，称为 **觉醒星级**，也是 0~5 星。

觉醒星级的效果：

1. **属性伤害强化**：对精灵主副属性技能做伤害强化加成
   - 例：罗隐为地+恶属性，觉醒星级会提升「地系伤害强化」和「恶系伤害强化」
   - 具体每级数值待补充

2. **系别极化**：提升精灵攻击造成克制时的伤害，同时减弱对手抵抗精灵攻击时的效果（攻击增强）
   - 具体每级数值待补充

3. **系别钝化**：降低精灵被克制时受到的伤害，并提升精灵抵抗攻击时的效果（防御增强）
   - 具体每级数值待补充

### 完整星级体系总结

```
精灵星级 = 普通五星（成长值加成）+ 觉醒五星（属性伤害强化 + 系别极化/钝化）
```

- 普通星级：纯数值提升（六维成长值）
- 觉醒星级：战斗机制增强（属性伤害 + 克制/抵抗效果）
