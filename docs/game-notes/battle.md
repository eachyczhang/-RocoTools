# 战斗系统

## 背包

- 每个洛克（角色）拥有 **3 个背包**
- 每个背包可携带 **6 只精灵**
- 每个背包可携带 **1 个道具**（也可不携带）

## 道具

目前有两种背包道具：

### 愿力强化

- **使用对象**：非污染、非奇异、非首领血脉的精灵
- **效果**：使用后，该精灵的第一个技能在**当前回合**变更为「愿力冲击」
- **愿力冲击属性**：
  - 攻击属性 = 精灵当前**血脉属性**（非本体属性）
    - 例：火神本体火属性，若修改为机械血脉 → 愿力冲击为机械属性
  - 物攻/魔攻 = 自动取精灵物攻、魔攻**更高的一项**适配
  - 可**应对状态技能**，应对成功后威力变为 **×1.5**
- **具体数值/公式**：待补充

### 进化之力

- **使用对象**：首领血脉的精灵
- **效果（最高进化形态）**：变身为**首领形态**
  - 维持整场战斗
  - 拥有更高的属性
  - 拥有升级版特性
- **效果（非最高形态）**：进化为**高一级形态**
  - 不会获得升级版特性
