# 能量系统

## 基础规则

- 每只精灵拥有 **10 点能量**（魔力值），入场时满额
- 能量上限固定为 10，当前版本不可突破
- **无自然回复**，能量管理完全依赖技能和聚能

## 聚能

- 所有精灵通用的**状态技能**
- 回复 **5 点**能量
- **占据一个回合行动**
- 属于状态技能，可被应对状态的技能应对

## 能量变化方式

| 方式 | 说明 |
|------|------|
| 技能消耗 | 使用技能扣除对应 cost 能量 |
| 聚能 | +5 能量，占一回合 |
| 回复类技能 | 部分技能可回复自身/同伴/后台精灵能量 |
| 减耗类技能 | 部分技能可减少某技能的能量消耗 |
| 回合结束效果 | 部分状态/技能在回合结束时减少能耗（临时或永久） |

## 数据字段

- `cost`：技能的能量消耗值
- cost = 0 表示不消耗能量
