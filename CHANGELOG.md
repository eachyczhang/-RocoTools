## V1.1 — 用户反馈模块 + CDN 加速 (2026-05-30)

> 新增用户反馈系统，支持用户端提交反馈（含图片）和管理端审阅处理。配置腾讯云 CDN 全站加速。

### 💬 用户反馈模块

#### 用户端（FeedbackFAB 浮动按钮）

- 全站右下角浮动反馈按钮（FAB），支持展开/收起
- 反馈类型：Bug报告 / 功能建议 / 其他
- 内容输入：10-500字，支持联系方式（选填）
- 图片上传：最多2张，单张≤3MB，自动压缩为WebP（lossless，最大1920px）
- 自动采集环境信息：页面URL、页面标题（中文映射）、设备类型、屏幕尺寸、UA、暗色模式
- 提交冷却：可配置冷却时间（管理端设置），前端倒计时显示
- IP频率限制：每IP每小时最多10次
- 功能开关：管理端可全局开启/关闭反馈入口

#### 管理端（/admin/feedbacks）

- 反馈列表：分页 + 状态筛选（待处理/已读/已解决/已忽略）+ 类型筛选
- 状态统计：顶部Tab显示各状态数量
- 反馈详情：展开查看完整内容、图片预览、环境信息、管理员备注
- 操作：修改状态、添加备注、删除反馈（含图片清理）
- 冷却时间配置：可设置用户提交冷却秒数（弹窗反馈保存结果）
- 功能开关：一键开启/关闭用户反馈功能
- Dashboard统计：待处理数、本周新增、总数

#### 技术实现

- 后端：`routes/feedbacks.js`（用户端）+ `routes/admin/feedbacks.js`（管理端）
- 前端：`FeedbackFAB.vue`（浮动按钮组件）+ `AdminFeedbacks.vue`（管理页面）
- 数据库：`feedbacks` 表（自动建表，含索引）
- 图片存储：`/uploads/feedbacks/{YYYY-MM}/` 按月归档
- 安全：XSS防护（sanitize）、文件类型白名单、大小限制、IP限流

### 🌐 CDN 全站加速

- 腾讯云 CDN 全站加速：`eachz.cn` CNAME → `eachz.cn.cdn.dnsv1.com`
- 无需代码改动，前端相对路径自动经过 CDN
- 缓存规则：图片30天、JS/CSS 7天、API/HTML不缓存
- 404 状态码不缓存（防止缓存错误的 404 响应）
- SSL 证书：TrustAsia DV TLS RSA CA 2025（有效期至 2026-08-26）

## 版本管理规范

### 版本号规则

- **V{主版本}.{次版本}**
- 主版本 (Major)：重大里程碑、架构重构、不兼容变更
- 次版本 (Minor)：新功能模块、重要优化

### 发布流程

1. 确认所有功能已完成并测试
2. 更新 `CHANGELOG.md`（添加新版本section）
3. 更新 `.ai-memory.md`（同步最新状态）
4. 更新 `DOC_RULES.md`（整理时间）
5. Git 提交：`git commit -m "release: V{版本号} - {简要描述}"`
6. Git 推送，服务器自动部署

### 记录格式

每个版本包含：
- 版本号 + 发布日期
- 一句话概述
- 按模块分类的变更列表（使用 emoji 分类标识）
