# 赛季系统

## 赛季周期

- 游戏按赛季更新（S1、S2...）
- 每个赛季有独立的限定内容

## 通行证精灵

- 每赛季推出通行证，含 **2 只**通行证精灵可选
- 获取方式：
  - 自己拥有的果实
  - 去拥有果实的好友世界，通过重复捕捉概率获取

## 赛季限定精灵

- 每赛季有 **8 只**只能在当前赛季捕捉的精灵
- 这 8 只**必定拥有异色**版本

## 赛季常态异色

- 每赛季有 **8 只**随机的日常可捕捉精灵，在当前赛季拥有异色版本

## 赛季切换规则

- 赛季结束后，上述 16 只精灵（8 赛季限定 + 8 常态异色）：
  - **异色版本**除孵蛋外**无法再获取**
  - 精灵本体（非异色）正常可获取（限定精灵除外）

## 数据结构（管理端配置）

| 字段 | 说明 |
|------|------|
| season_id | 赛季标识（如 S1、S2） |
| pass_pets | 通行证精灵（2只 uid） |
| season_pets | 赛季限定精灵（8只 uid，必有异色） |
| shiny_pets | 常态异色精灵（8只 uid，本赛季限定异色） |
