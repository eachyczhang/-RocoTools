# 属性系统

## 基础规则

- 共 **18 种属性**
- 精灵可拥有 **双属性**：主属性 + 副属性（副属性可为空）

## 克制倍率

| 关系 | 倍率 |
|------|------|
| 克制（strong） | ×2 |
| 抵抗（resist） | ×0.5 |
| 双重克制（两属性均被克制） | ×3（非 ×4） |
| 双重抵抗（两属性均抵抗） | ×0.25 |

## 双属性计算

攻击属性 A 打防御方（主属性 E1 + 副属性 E2）：
1. 分别计算 A 对 E1、A 对 E2 的单属性倍率
2. 两个倍率相乘得到最终倍率
3. 特殊修正：2×2=3（双重克制封顶 ×3）

## 免疫

- 部分属性存在免疫效果（immunity 字段）
