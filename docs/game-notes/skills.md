# 技能系统

## 技能分类

| 分类 | 说明 |
|------|------|
| 物攻 | 物理伤害技能 |
| 魔攻 | 魔法伤害技能 |
| 防御 | 防御类技能 |
| 状态 | 状态类技能（如聚能） |

## 技能属性

- 每个技能有属性归属
- 消耗能量值（cost 字段）
- 固定威力值（power 字段，部分技能无固定威力）

## 学习方式

| 方式 | 说明 |
|------|------|
| 自然升级 | 精灵升级到指定等级自动学习 |
| 血脉学习 | 通过血脉系统解锁 |
| 技能石学习 | 使用技能石习得 |

## 应对机制

- 技能存在「应对」关系
- 状态技能可被应对状态的技能克制
- 聚能作为状态技能，也在被应对范围内

## 血脉系统

每只精灵拥有一种血脉，血脉决定可学习的血脉技能。

### 普通血脉

- 可自由修改为其他属性血脉
- 修改后解锁对应属性的血脉技能

### 特殊血脉（3 种）

#### 1. 污染血脉

| 项目 | 说明 |
|------|------|
| 限制 | 不可学习技能、不可使用技能石、不可直接修改血脉 |
| 技能 | 随机拥有 4 个技能 |
| 净化 | 使用药水净化 → 变为**奇异血脉** |
| 净化后 | 保留污染时 4 个技能中的 1 个作为血脉技能 |

#### 2. 奇异血脉

| 项目 | 说明 |
|------|------|
| 来源 | 污染血脉净化后获得 |
| 技能 | 拥有 1 个随机血脉技能（净化保留） |
| 修改 | 修改血脉后改为对应血脉的可学习技能 |

#### 3. 首领血脉

| 项目 | 说明 |
|------|------|
| 条件 | 部分精灵特有 |
| 效果 | 配合「进化之力」道具可进化/变身为首领形态 |
| 首领形态 | 更高属性 + 升级版特性，维持整场战斗 |

### 血脉与道具的交互

| 血脉类型 | 愿力强化 | 进化之力 |
|---------|:---:|:---:|
| 普通血脉 | ✅ 可使用 | ❌ |
| 污染血脉 | ❌ | ❌ |
| 奇异血脉 | ❌ | ❌ |
| 首领血脉 | ❌ | ✅ 可使用 |
