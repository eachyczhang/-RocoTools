# 皮卡月刊（角色时装）

## 概念

皮卡月刊是洛克王国世界的角色时装系统，每期时装会绑定多个精灵，并有特殊的出场动画。

- **绑定精灵**：每期时装可关联多个精灵，每个精灵有一男一女两个洛克时装
- **出场动画**：穿戴时装时有特殊精灵出场效果
- **概念图**：每期月刊只有一张总概念图，包含所有精灵时装的合照

## 时装分类

| 类型 | 说明 |
|------|------|
| 洛克男 | 某精灵的男性角色时装图片 |
| 洛克女 | 某精灵的女性角色时装图片 |
| 概念图 | 该期月刊所有时装的总概念宣传图（一期一张） |

## 期数标识

- 格式：`YYYYMM`（年月）
- 示例：`202605` 表示 2026 年 5 月

## 售卖周期

- 每期月刊有**上架日期**和**下架日期**
- **上架日期**：该期月刊开始售卖的时间
- **下架日期**：该期月刊结束售卖的时间，过期后只能等返厂或入库
- 下架后自动上架下一期月刊

## 数据存储

- 主表：`pika_monthlies`（期数、名称、概念图、时间、排序）
- 关联表：`pika_monthly_pets`（月刊-精灵多对多，每个精灵存洛克男/洛克女图片）
- 主键：`pika_monthlies.id` 自增
- 关联：`pika_monthly_pets.pet_uid` → 精灵表
