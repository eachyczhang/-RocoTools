# 活动系统

> 每个赛季开始时，官方会提前公布整个赛季的活动时间节点。
> 活动分为「版本活动」和「常驻课题」两大类。

---

## 版本活动

### 1. 主推版本活动（贯穿全赛季）

- 每赛季 1-2 个，贯穿整个赛季周期
- 通常为赛季主题故事线/大型玩法
- 示例：S2「呱呱上学记」(05.21-07.15)、「主题摄影评选赛」(05.23-07.15)

### 2. 领地试炼（每赛季 2-3 个）

- 每赛季安排 2-3 次，每次持续约 3 周
- 副本挑战类玩法
- 示例：S2「领地试炼-深渊罗隐」(05.29-06.18)、「领地试炼-高帽酿者」(06.26-07.16)

### 3. 精灵抱抱团

- 指定精灵的限时捕捉活动
- 约持续 1 周
- 与大世界观察日记**交替出现**，穿插在赛季中
- 示例：S2 精灵抱抱团 (05.29-06.04)、(06.26-07.02)

### 4. 大世界观察日记

- 捕捉满足特定条件（身高/体重/性格）的精灵
- 约持续 1 周
- 与精灵抱抱团**交替出现**
- 示例：S2 大世界观察日记 (06.12-06.18)、(07.10-07.16)

### 5. 大量出没（每周末）

- 每周的 **周五、周六、周日** 有特定精灵大量出现
- 一般为该赛季会拥有异色的 8 只精灵轮换
- 每次大量出没持续 3 天（周五-周日）
- 示例：S2 大量出没-黑鸦町 (05.22-05.24)、大量出没-公平岛 (06.05-06.07)

## 常驻课题

### 1. 命定花种

- **周期**：每期持续 2 周
- **内容**：固定性格、自带三级亲密度的精灵
- **玩法**：单人挑战，击败后解锁精灵特殊成就
- **精灵来源**：与当期「皮卡月刊」时装对应的多只精灵
- **精灵特性**：
  - **固定性格**：命定花种精灵的性格是固定的（非随机）
  - **三级亲密度**：捕捉后必定自带三级亲密度
  - **天赋保底**：必然拥有三个天分加成，即必然为「相当好天分」或「了不起天分」
- **固定技能**：
  - **愿力冲击**：所有命定花种精灵必定拥有。攻击属性 = 精灵血脉属性（即主属性），自动适配物攻/魔攻更高的一项，可应对状态技能（应对成功威力 ×1.5）
- **技能配置**：每只命定花种精灵最多拥有 4 个技能（含愿力冲击），其余 3 个可从升级技能、血脉技能、技能石技能中配置（管理端 /admin/fate-flower）
- **性格配置**：每只命定花种精灵的固定性格可在管理端单独配置（管理端 /admin/fate-flower）
- 示例：S2 命定花种 05.21-06.04、06.05-06.18、06.19-07.02、07.03-07.15

### 2. 星光对决

- **周期**：每周更新
- **内容**：PVE NPC 对战
- **奖励**：赛季通行证硬币，用于兑换资源
- 示例：S2 星光对决 每周一期（05.25-05.31、06.01-06.07...）

### 3. 皮卡摄影委托

- **周期**：每期持续 2 周（与命定花种同步）
- **内容**：给皮卡月刊对应的精灵拍照
- **奖励**：完成全部拍摄后赠送家园家具（壁画）
- **精灵来源**：与当期「皮卡月刊」精灵一致
- 示例：S2 皮卡摄影委托 05.21-06.04、06.05-06.18、06.19-07.02、07.03-07.15

## 活动节奏规律（以 S2 为例）

```
版本活动：
├── 主推活动 ──────────────────────────── 贯穿全赛季
├── 领地试炼 ─────── ×2-3，各约3周
├── 精灵抱抱团 / 大世界观察日记 ──── 交替出现，各约1周
└── 大量出没 ─────── 每周末（周五-周日），8只异色精灵轮换

常驻课题：
├── 命定花种 ─────── 每2周一期，跟随皮卡月刊
├── 星光对决 ─────── 每周一期
└── 皮卡摄影委托 ── 每2周一期，跟随皮卡月刊
```

---

## 活动与皮卡月刊的关联

- 「命定花种」和「皮卡摄影委托」的精灵 = 当期皮卡月刊时装对应的精灵
- 三者周期同步（均为 2 周一期）
- 数据管理时，皮卡月刊配置完精灵后，命定花种和皮卡摄影委托可关联相同的精灵列表

## 数据结构（管理端配置）

活动存储在 `season_events` 表中：

| 字段 | 说明 |
|------|------|
| season_id | 所属赛季（如 S2） |
| category | 分类：`version`（版本活动）/ `mass_outbreak`（大量出没）/ `routine`（常驻课题） |
| sub_type | 常驻课题子类型：`fate_flower`（命定花种）/ `star_battle`（星光对决）/ `pika_photo`（皮卡摄影委托） |
| name | 活动名称 |
| start_date | 开始日期 |
| end_date | 结束日期 |
| pet_uid | 关联精灵（大量出没用） |
| periods | JSON 数组，常驻课题的多段时间 |
| image | 活动封面图 |
| row_order | 排序 |
